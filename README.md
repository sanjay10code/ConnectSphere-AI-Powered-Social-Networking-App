# ConnectSphere-AI-Powered-Social-Networking-App
Developed a full-stack social networking mobile application inspired by modern social media platforms, enabling users to create profiles, share posts, upload images/videos, follow users, and interact through likes, comments, and real-time messaging.

Built scalable backend APIs and secure authentication systems using React Native, Node.js, Express.js, and MongoDB to support seamless user experiences and high-performance data handling. Implemented personalized content feeds, push notifications, and live chat functionality to enhance user engagement and social interaction.

Integrated AI-powered recommendation features and smart content filtering mechanisms to deliver customized user experiences while ensuring scalable and responsive mobile application performance.
